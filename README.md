# Gestor de Solicitudes

> Un sistema diseñado para facilitar la gestión de solicitudes dentro de una organización. Este proyecto es ideal para optimizar procesos administrativos y mejorar la comunicación entre departamentos.

## Características principales
- **Gestión eficiente** de solicitudes.
- **Notificaciones** automáticas sobre el estado de cada solicitud.
- **Interfaz intuitiva** para usuarios.
- **Reportes detallados** para análisis.
- **Soporte para múltiples usuarios.**

## Uso
1. Accede a la aplicación en [http://localhost:3000](http://localhost:3000) (o la dirección configurada).
2. Registra una solicitud utilizando el formulario.
3. Revisa el estado y notificaciones en el dashboard.
